# Visitas Técnicas SENAI Bahia — versão com dados reais

Esta versão consolida os 19 relatórios Excel e a apresentação executiva.

## Regras aplicadas
- `PARTICIPANTES` = quem visitou/participou, conforme o cabeçalho de cada aba.
- `ACOMPANHADO POR` = quem acompanhou cada item durante a visita.
- Ano 2025 normalizado para 2026, conforme orientação do responsável.
- VCA normalizado para Vitória da Conquista.
- Abas com unidade interna divergente foram excluídas da unidade incorreta.
- Avatares associados apenas quando houve correspondência nominal.

## Fluxo
1. Clique na unidade ou cidade.
2. O mapa aproxima e abre a ficha.
3. Veja quem visitou e quem acompanhou.
4. Clique em um participante.
5. Escolha a área no dropdown.
6. Consulte principais pontos, boas práticas, oportunidades, observações, responsáveis, prazos e itens técnicos.

## Publicação
Substitua os arquivos do repositório e faça commit na branch `main`.
